Here are some of the in class projects I have made for my web development class!

ICE stands for In Class Exercise, and these are projects that I have made during class time to practice different web development techniques and concepts.

Click on the images to view the projects!

Prophet Card

This is an in class exercise where we practiced cloning a wireframe. We created a trading card style item with a photo of President Nelson, along with a quote and some other elements.

Apostle Spotlight

For this project, I created a web page that shows the twelve apostles of the church of Jesus Christ of Latter-day Saints. It includes elements of responsive web design, and will change as you resize the browser. It will also change styles when you mouse over each Apostle.

Favorite Devotional Table

This is a simple table that I created to learn more about implementing tables in HTML. It also includes a form at the bottom.

Valentine's Day Page

This is a Valentine's day website I made to practice CSS styling of web pages. We were given the HTML and only able to change the CSS.

Country Flags with Grid

This is a project where I created country flags using only CSS Grid. It was a good exercise in responsive design and CSS layout.

CSS Positioning Activity

For this project, I was given code including several different squares and a sample image to match them. Only editing the CSS, I was able to match the given squares to the sample image. I learned a lot about CSS positioning and layout techniques using grid.

Image Gallery Issues

For this project, we were given an image gallery that took a very long time to load, and we were tasked with making it load more quickly. This is an important aspect of web development, and I learned about optimizing image loading and performance.
